# **Power Automate > Formulas**

## **Expresiones | String function | toLower**

Convierte una texto a minúsculas.

El resultado de la expresión es un **"texto"**.

### Sintaxis

```
toLower(text)
```

----

**<u>Ejemplo 01</u>**

#### **Input**

<table>
    <thead>
        <tr>
            <th>Nombre</th>
            <th>Valor</th>
            <th>Tipo</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>text</td>
            <td>'Power Automate'</td>
            <td>Texto</td>
        </tr>
    </tbody>
</table>

```
toLower(text)
```

#### **Output**

```
'power automate'
```
