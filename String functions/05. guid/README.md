# **Power Automate > Formulas**

## **Expresiones | String function | guid**

Genera una cadena globalmente única (GUID).

El resultado de la expresión es un **"texto"**.

### Sintaxis

```
guid()
```

----

**<u>Ejemplo 01</u>**

#### **Input**

```
guid()
```

#### **Output**

```
'7aab47a0-bf0b-4c27-933b-d9f1f1845dba'
```
